## 设置皮肤教程

> **注意**
> 因为权限API的问题，无法开放skin指令权限。**下列教程已失效。**
> 你可以使用[SkinShuffle](https://www.mcmod.cn/class/13509.html)更换皮肤。*（应该可以，我没测试）*
{.is-warning}

如果你为正版玩家，应该无需设置皮肤；如果进游戏后皮肤消失可以使用`/skin set mojang <你的正版id>`来设置皮肤。
同样地，离线玩家也可以使用上面那个指令来设置皮肤，只需要填写你想要设置皮肤的正版id就能把他的皮肤偷过来。
例如`/skin set mojang hsds`，这样你就~~可以把黑山大叔的皮肤偷过来了(~~
如果你想要自己设置皮肤也可以。首先，你需要寻找一个图床网站，**最好找没有文件有效期限制的，并且一定要在本地备份**，将你想要的皮肤上传至图床中，然后复制网站给你的图片URI，进入游戏。如果你的皮肤为粗壮手臂(steve)，请使用`/skin set web classic <"皮肤链接">`，**注意链接应该用英文双引号括起来**；如果你的皮肤为纤细手臂(alex)，则使用`/skin set web slim <"皮肤链接">`。**同样地，链接应该用英文双引号括起来**。如果你不知道应该使用哪个指令，请分别尝试，然后检查皮肤（特别是手臂）是否出现错误。
如果你是在网络上寻找皮肤，请不要直接使用搜索引擎给你提供的图片，这通常格式极其糟糕。Minecraft JAVA版原版最高仅支持64\*64像素的皮肤。如果想使用更高分辨率的皮肤，请自行寻找其它mod。
